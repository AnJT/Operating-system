## 一、项目概述

本项目为一个内存调页模拟程序。在本项目模拟的系统中，执行的指令共有320条，一个页面可以存放10条指令，所有指令分为32页，系统中共有4个页框可供程序使用。本项目将以可视化的方式展示程序执行过程张总的换页过程，以及缺页率的统计。

## 二、功能说明

本项目实现了如下功能：

- 查看下一条指令地址
- 缺页数统计
- 缺页率统计
- 换页提示
- 指令命中提示
- 指令执行数据统计
- 单步执行指令
- 连续执行指令
